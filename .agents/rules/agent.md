# standards
- Clean standard code HTML5 and standard open source CSS
- Simple easy to edit or understand coding 
- When committing changes in a host-specific website subfolder, start the git commit subject with the exact host name followed by ` - `.
- Required format: `<hostname> - <short summary>`.
- Example: `oakland.center - update hero copy`
- Keep the host prefix consistent with the folder/domain name so commits are easy to search and group by site.
